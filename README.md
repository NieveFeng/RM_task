## 任务1_Armor

**<u>主程序存放于armor.cpp</u>**

1. 运用最传统的算法，spilt()对源图像分离成三通道，再用通道2减去通道0得到红色灯条

2. 二值化图像代入寻找轮廓的函数findcontours()
3. 遍历轮廓，找出灯条的特征——筛除条件：area轮廓的面积

## 任务2_Rune

**<u>主程序存放于rune.cpp</u>**

5. 绘制最小包围旋转矩形and能量机关的击打点
